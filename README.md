# Real-Time Collaborative Project Management System

A full-stack project management platform that allows teams to collaborate, manage tasks, and track project progress in real time.

## 🚀 Tech Stack

Frontend:
- React
- JavaScript
- Vite

Backend:
- Python
- Flask
- REST API

Database:
- PostgreSQL

Version Control:
- Git
- GitHub

---

## 📌 Features

- User Authentication (Register & Login)
- Project creation and management
- Task assignment and tracking
- Dashboard for project overview
- Real-time team collaboration

## ⚙️ Installation

Clone the repository:

git clone https://github.com/Sunlight8169/real-time-project-management-system.git

Go to project directory: 

cd real-time-project-management-system

Install frontend dependencies:
cd frontend
npm install
npm run dev
